# Indicium Event Manager API.

Install packages
```bash
mvn clean install
```

Run application
```bash
mvn spring-boot:run
```

Payload to create event
```json
{
    "title": "",
    "description": "",
    "startDate": "2019-11-07T23:00:00.000+0000",
    "endDate": "2019-11-07T23:00:00.000+0000",
    "slug": "",
    "status": "draft",
    "location": ["52.0841868", "5.0824915"],
    "url": "https://noudadrichem.com",
    "categories": ["", ""]
}
```
